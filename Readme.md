# MyGinProject

一个基于 [Gin](https://github.com/gin-gonic/gin) 的 Web API 项目。

## 🚀 功能特性

- [x] Gin HTTP 框架
- [x] GORM 操作 MySQL
- [x] Viper 配置管理
- [x] JWT 用户认证
- [x] Bcrypt 密码加盐存储
- [x] 统一的 Response 封装
- [x] 优雅关闭 (Graceful Shutdown)
